# 환율 예측 서비스 — 디자인 시스템

> Phase 5: Design System
> 프로젝트 전반에 사용하는 공통 디자인 규칙과 컴포넌트 정의

---

## 1. 디자인 토큰 (Design Tokens)

> **디자인 토큰이란?** 색상, 간격, 글자 크기 등 디자인의 기본 재료입니다.
> LEGO 블록처럼 이 토큰들을 조합해서 화면을 만듭니다.

### 색상 (Colors)

| 토큰 이름 | 값 | 용도 |
|----------|-----|------|
| `--color-primary` | `#4f46e5` | 메인 색상 (보라) — 헤더, 버튼, 강조 |
| `--color-up` | `#10b981` | 환율 상승 — 초록 |
| `--color-down` | `#ef4444` | 환율 하락 — 빨강 |
| `--color-bg` | `#f8fafc` | 페이지 배경색 |
| `--color-card` | `#ffffff` | 카드 배경색 |
| `--color-text` | `#1e293b` | 기본 텍스트 |
| `--color-sub` | `#64748b` | 보조 텍스트 (날짜, 설명 등) |

### 간격 (Spacing)

| 이름 | 값 | 예시 |
|------|-----|------|
| 작음 | `8px` | 카드 내부 작은 여백 |
| 보통 | `16px` | 기본 패딩 |
| 큼 | `24px` | 섹션 내부 여백 |
| 매우 큼 | `40px` | 섹션 간 여백 |

### 모서리 둥글기 (Border Radius)

| 이름 | 값 | 적용 대상 |
|------|-----|----------|
| `--radius` | `16px` | 카드, 예측 박스 |
| 배지 | `99px` | "상승 예측" 뱃지 (완전히 둥글게) |

### 그림자 (Shadow)

```
--shadow: 0 4px 20px rgba(0,0,0,0.08)
```
카드처럼 살짝 떠 있는 느낌을 줍니다.

## 2. 공통 컴포넌트 (Components)

> **컴포넌트란?** 재사용 가능한 화면 조각입니다.
> 레고 블록처럼 필요한 곳에 끼워 쓸 수 있어요.

### 환율 카드 (RateCard)

```
┌─────────────────────────┐
│ 미국 달러 → 한국 원       │  ← currencies (보조 텍스트)
│ (USD/KRW)               │
│                         │
│  1,320.5  KRW           │  ← 큰 숫자 + 단위
│                         │
│ 기준일: 2026-03-05       │  ← date (작은 보조 텍스트)
└─────────────────────────┘
```

**HTML 클래스:** `.rate-card`
**사용 위치:** 오늘의 환율 섹션

---

### 예측 카드 (PredictionCard)

```
┌──────────────────────────────────────────┐
│  7일 후 예측 환율          예측 신뢰도      │
│  (USD/KRW)                              │
│                                          │
│  ▲ 1,335 KRW    72%                     │
│  [상승 예측]     ████░░  AI 모델 기반      │
└──────────────────────────────────────────┘
```

**HTML 클래스:** `.prediction-card`
**상태 variants:**
- `up` — 초록색 (상승)
- `down` — 빨간색 (하락)

---

### 뱃지 (Badge)

작은 라벨 표시용 컴포넌트

| 종류 | 색상 | 사용 예시 |
|------|------|---------|
| `badge-up` | 초록 배경 | "상승 예측" |
| `badge-down` | 빨강 배경 | "하락 예측" |

---

### 신뢰도 바 (ConfidenceBar)

```
████████░░░░  72%
```

**HTML 클래스:** `.confidence-bar` / `.confidence-bar__fill`
- 전체 너비의 `confidence`% 만큼 채움
- 색상: `--color-primary` (보라)

---

### 차트 (Chart)

Canvas API로 그린 선 그래프

| 요소 | 설명 |
|------|------|
| 선 색상 | `#4f46e5` (보라) |
| 면적 채우기 | 보라 그라디언트 (투명도 적용) |
| 포인트 | 보라 원 + 흰 테두리 |
| 그리드 | 점선 (`#e2e8f0`) |
| X축 | 날짜 (MM-DD 형식) |
| Y축 | 환율 숫자 |

## 3. 타이포그래피 (Typography)

| 용도 | 크기 | 굵기 |
|------|------|------|
| 로고 | 24px | 700 (Bold) |
| 섹션 제목 | 18px | 600 (SemiBold) |
| 환율 큰 숫자 | 28px | 700 (Bold) |
| 예측 큰 숫자 | 36px | 700 (Bold) |
| 기본 텍스트 | 16px | 400 (Regular) |
| 보조 텍스트 | 13px | 400 (Regular) |
| 날짜/캡션 | 12px | 400 (Regular) |

**글꼴:** `Apple SD Gothic Neo`, `Noto Sans KR`, `sans-serif`
(한글 웹폰트 기본값 사용)
